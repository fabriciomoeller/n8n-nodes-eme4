# npm install



## pnpm
package manager for n8n custom nodes

```
npm install -g pnpm
npm install -g rimraf
```


## rimraf
The UNIX command rm -rf for node in a cross-platform implementation
```
npm install -g rimraf
```


## use pnpm to install the packages
In the root folder

```
pnpm install
```

node_modules folder will be created
